# Bots Dashboard Feature Roadmap

## Core principles

- Modern, sleek, dark-purple UI.
- Fast role-based workflows for staff.
- Owner-level global control.
- API-first backend integrations.

## A. Authentication and Access

- Discord OAuth login.
- Owner override access.
- Tier-based permissions (helper to owner).
- Per-page authorization gates.
- Session expiration and secure cookies.
- Audit trail for sensitive actions.

## B. Global Dashboard

- Fleet status cards for all six bots.
- Open and closed ticket counters.
- Member and presence overview.
- Notification center for incidents.
- Global search for users, tickets, and configs.

## C. Server Management

- Members browser with role and join data.
- Roles overview and hierarchy visualization.
- Channel and category inventory.
- Discord audit summaries.
- Scheduled events and invite insights.

## D. Tickets Bot Features

- Ticket table with advanced filters.
- Ticket detail pages and transcript links.
- Blacklist management and expiry handling.
- Ticket type configuration controls.
- Staff ticket analytics and SLAs.

## E. Management Bot Features

- Ban and timeout control center.
- Media rank workflow panel.
- YouTube checker status and controls.
- Mention protection analytics.
- Prebuilt SQL report views.

## F. Utilities Bot Features

- Account sync visibility and tools.
- Poll creation and lifecycle controls.
- Tags CRUD interface.
- Screenshare operations dashboard.
- Application review workflows.
- Helper response editor.

## G. Staff Bot Features

- Staff leaderboard and trend charts.
- Strike report queue and votes.
- Trainer and trainee management.
- Data export and wipe controls.
- Staff logs and formatting insight views.

## H. Leader Bot Features

- Faction registry and editing.
- Thread request tracking.
- Bundle ticket controls.
- Close/lock/rename workflows.
- Telegram-linked request timeline.

## I. Games Bot Features

- XP and level profile views.
- Leaderboard controls.
- Monthly wipe workflow.
- Chat and DM game configuration panels.
- Achievement and milestone management.
- Counting channel operations.

## J. Fleet Operations (Phase 2+)

- Process status (up/down/degraded).
- Start/stop/restart actions.
- Live logs and error alerting.
- Cog management.
- Hot config reload.

## K. Analytics and Reporting

- Cross-bot trend dashboards.
- Read-only custom query builder.
- CSV/JSON exports.
- Scheduled report delivery.

## L. Integrations

- Telegram status integration.
- Google Sheets integration health.
- YouTube quota and job status.
- Minecraft API control panel.

## M. UX and Productivity

- Command palette.
- Keyboard shortcuts.
- Mobile-optimized dashboard shell.
- Favorites and recent pages.
- Accessible, animated UI components.
